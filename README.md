# Business Performance Analytics Dashboard

## Overview
This project analyzes product performance and user engagement using SQL, Python, and Tableau to support data-driven decision-making.

## Business Problem
Stakeholders need visibility into key performance metrics, user engagement trends, and product performance to identify growth opportunities and inefficiencies.

## Tools Used
- SQL
- Python (Pandas, NumPy)
- Tableau

## Key Features
- KPI tracking dashboard
- User engagement analysis
- Product performance trends
- Interactive filters and drill-down views

## Key Insights
- Identified high-performing products contributing to revenue growth
- Detected engagement drop-offs in specific user segments
- Highlighted trends supporting strategic decision-making

## Project Structure
- `/data` → raw and processed datasets  
- `/notebooks` → Python analysis  
- `/sql` → SQL queries  
- `/images` → dashboard screenshots  

## Outcome
This project demonstrates end-to-end data analysis, dashboard development, and business insight generation.
